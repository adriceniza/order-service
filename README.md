# Order Service

The goal of this repository is not to build a complete e-commerce platform, but to explore backend engineering practices commonly used.

## Engineering Practices

- Test-Driven Development (TDD)
- Domain-Driven Design (DDD)
- Clean Architecture
- RESTful APIs
- Domain Events
- Idempotent Operations
- PostgreSQL Transactions
- Observability
- CI/CD
- Conventional Commits

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker
- Pytest
- Prometheus
- Grafana
- GitHub Actions
